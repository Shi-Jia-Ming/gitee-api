## 描述

本项目封装了 Gitee API 的功能，使得调用者可以通过库中的函数访问 Gitee 服务。

本项目基于 TypeScript。项目主要使用 axios 库进行 HTTP 请求。你可以参考 [Gitee API](https://gitee.com/api/v5/swagger) 了解请求的详细信息。
